# 115-PCIC — 115年品管土建題庫練習網站

單一 HTML 檔的題庫練習網頁應用，收錄 **115 年品管（土建類）題庫共 2,689 題**。

## 特色

- 純前端、零依賴：只有一個 `index.html`，題庫資料內嵌，開啟即用
- 支援手機瀏覽（RWD、`viewport-fit=cover`）
- 作答即時判定：正確／錯誤即時標色回饋
- 繁體中文介面，適配 Noto Sans TC / 微軟正黑

## 使用方式

直接用瀏覽器開啟 `index.html` 即可，不需要伺服器、不需要安裝任何東西。

也可以部署到 GitHub Pages：Settings → Pages → 選擇 `main` 分支即可獲得線上網址。

## 檔案結構

```
115-PCIC/
└── index.html   # 完整應用（UI + 題庫資料 + 判定邏輯）
```
